# Convert RandomSystemState to Data Frame

Converts a `RandomSystemState` object into a data frame containing
overall and block-wise treatment arm filling levels.

## Usage

``` r
# S3 method for class 'RandomSystemState'
as.data.frame(x, ...)
```

## Arguments

- x:

  A `RandomSystemState` object.

- ...:

  Additional arguments (currently unused).

## Value

A data frame with columns for overall and block-wise treatment arm
filling levels.
